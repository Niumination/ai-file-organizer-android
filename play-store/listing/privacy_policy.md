# Privacy Policy – AI File Organizer v1.1

Effective: 24 June 2026
Developer: Niumination
Package: com.arena.aifileorganizer

AI File Organizer is open source and respects your privacy.

1. No data collection by us. No analytics, ads, trackers.
2. Gemini API Key: you provide your own Google Gemini API key. Stored encrypted on-device (EncryptedSharedPreferences AES-256-GCM). Never sent to our servers – only directly to generativelanguage.googleapis.com via HTTPS.
3. File content: App reads ONLY the folder you explicitly pick via Android SAF. Text preview max ~600 chars per file may be sent to Gemini for categorization. Files are NEVER uploaded in full. Images OCR 100% on-device (ML Kit).
4. Third parties: Google Gemini API – https://ai.google.dev/terms • ML Kit – on-device.
5. Permissions: only android.permission.INTERNET and android.permission.VIBRATE (haptic). Storage via user-granted SAF.
6. Open source: https://github.com/niumination/ai-file-organizer-android
7. Contact: GitHub Issues
